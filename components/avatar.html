Avatars
An avatar is a graphical representation of a user or the user's character or persona. With Silk UI use avatars in many amazing way.
Regular Avatar
Regular Avatar would be a rectangular image. Use "avatar" class on your image, it will make it an avatar!
Circular Avatars
In cases where circular avatars would suit your UI more, use "avatar-circular" class with regular avatar class to make your avatar circular
Avatar With Different Sizes
To provide your needs for different avatar sizes, you can choose from "avatar-xl", "avatar-large", "avatar-medium" and "avatar-small". You can use these classes with all kind of avatars (regular, circular, text-avatars)
Text Avatars
Text avatar would come in handy when there is no image available. You can use "avatar-circular" to make them circular. You can use it different size classes too.
